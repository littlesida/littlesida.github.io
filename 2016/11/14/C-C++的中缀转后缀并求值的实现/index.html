Description

输入一个中缀算术表达式S，S中的操作数为0到9，只含+，-和*，/运算，也可能含有括号()，运算符的计算顺序和实际四则运算的计算顺序相同. 请输出与S等价的后缀表达式，注意输出结果不要含有多余的括号或空格.

Input

输入有多组数据.

每组数据是一个中缀表达式S，S的长度不超过50. 输入的S保证合法，而且不包含多余的空格或制表符.

输入以#号结束.
