在windows下使用zip包安装MySQL5.7.16

  • MySQL
  • windows
  • 软件

在windows下安装MySQL常见有两种方式，第一种是使用zip包安装，即本文所讲方式；第二种为使用msi格式文件直接安装。
本文为在ubuntu下使用源文件安装失败（二进制文件安装成功）后使用windows尝试。
使用的环境为windows10 64位操作系统。

下载zip包

首先，到MySQL的官网下载相应的zip包，下载地址为 zip包下载地址

解压zip包

下载完成之后，右键解压zip包，并将解压出来的文件夹重命名为mysql-5.7.16，然后将该文件夹剪切到 D:\Program Files (x86)目录下(该路径为你想让mysql安装的路径)

配置环境变量

在开始安装之前，需要配置好环境变量。右键我的电脑→属性→高级系统设置→环境变量。找到系统变量中的path，点击“编辑”，将刚才mysql-5.7.16文件夹下的bin的路径加入到其中。由于win10中每个环境变量是分开的，所以直接新建一个即可，对于win7和win8的系统，需要先添加一个;后再将路径加在后面。（注意不要把原先的内容删除，是追加）

修改配置文件

配置完环境变量之后，需要先修改配置文件的信息。配置文件名为my-default.ini，该文件就在mysql-5.7.16目录下。打开该文件，可以看到在17-21行有basedir,datadir,port等信息。将basedir修改为mysql-5.7.16的路径，datadir为mysql-5.7.16路径再加上\data.Port端口设置为3306.并将前面的注释符号#删除。

安装MySQL

以管理员的身份运行cmd窗口，win10系统右键左下角的微软图标，可以看到命令提示符(管理员)这一项。然后cd进入到mysql-5.7.16的bin目录下。如果存放目录和我一样，使用d:切换到D盘，请注意Program Files (x86)需要用双引号括起来，且在s和(间存在一个空格。若没有管理员权限，则安装时会出现错误。

然后，输入mysqld -install安装。成功的话，会出现如下提示：

如果要删除，则使用 mysqld -remove命令

初始化数据库

安装完成之后，直接使用net start mysql或者mysqld –console启动时会出现错误提示，原因在于没有初始化数据库。在这之前，你可以查看mysql-5.7.16目录，此时，目录下是不存在data文件夹的。
使用mysqld –initialize 初始化数据库，完成之后，可以看到，在mysql-5.7.16目录下出现了data文件夹。

运行MySQL

使用net start mysql 来运行。启动成功后 输入mysql -u root -p来进入数据库。输入密码出直接按回车进入。（部分默认回车）。如果出现错误，则查看data文件夹下的.err文件，查看初始化数据库时产生的临时密码。在靠近上方的部分或者直接搜索err查找文件。

输入密码后进入成功，输入exit退出。

修改密码

到上一步位置，MySQL的安装算是完成，但是你会发现，使用时会出现1820的错误提示。让你修改密码才能执行命令。
1) set password = password(“新密码”);
2) alert user ‘root’@’localhost’password expire never;
3) flush privileges;
使用上面三个语句修改密码，其中，新密码为你想设置的新的密码，且注意没一句命令最后需要加英文分号（半角），然后exit退出即可。

至此，MySQL的安装过程全部完成，可以开心的使用MySQL了。
