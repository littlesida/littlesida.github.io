LeetCode NO.1 two Sum

Description

Given an array of integers, return indices of the two numbers such that they add up to a specific target.
You may assume that each input would have exactly one solution, and you may not use the same element twice.
Example:

分析

提要要求从给定的数组中找出两个数之和等于目标值。暴力法对此题目是会超时的。因此需要稍微简化一下。由于一定有解，且只有两个数的和，因此可以看成在数组中找到target减去某个值的结果。使用map，令值为key，index为map的value。使用find函数即可快速解决问题。
详细代码见github。

LeetCode NO.167 two Sum 2

Description

Given an array of integers that is already sorted in ascending order, find two numbers such that they add up to a specific target number.

The function twoSum should return indices of the two numbers such that they add up to the target, where index1 must be less than index2. Please note that your returned answers (both index1 and index2) are not zero-based.

You may assume that each input would have exactly one solution and you may not use the same element twice.

Input: numbers={2, 7, 11, 15}, target=9
Output: index1=1, index2=2

分析

此题于NO.1起始是一样的，只不过其下标不是从0开始，因此需要+1.
详细代码见github。

LeetCode NO.7 Reverse Integer

Description

Reverse digits of an integer.

Example1: x = 123, return 321
Example2: x = -123, return -321

Note:
The input is assumed to be a 32-bit signed integer. Your function should return 0 when the reversed integer overflows.

分析

此题于一般的数组翻转差不多，主要是需要添加判断翻转后是否溢出。因此可以在每次翻转一个数字后判断新的数字的最低位和刚才加上去的是否相等。
详细代码见github。

LeetCode NO.9 Palindrome Number

Description

Determine whether an integer is a palindrome. Do this without extra space.

分析

一道简单的判断是否为回文数的问题，在之前判断是否为回文数时总是把整个数字翻转后比较是否相等，今天在别人的答案里面看到了只需要翻转一半即可的答案。非常棒。
